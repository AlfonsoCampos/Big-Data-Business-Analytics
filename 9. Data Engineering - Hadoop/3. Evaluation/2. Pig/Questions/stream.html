STREAM

Envía los datos a un script o programa externo. 

Sintaxis

alias = STREAM alias [, alias …] THROUGH {`command` | cmd_alias } [AS schema] ;

Ejemplo de conjunto de datos desordenado:

A = LOAD 'data';
B = STREAM A THROUGH `stream.pl`;

Ejemplo de datos agrupados:

A = LOAD 'data';
B = GROUP A BY $1;
C = FOREACH B FLATTEN(A);
D = STREAM C THROUGH `stream.pl`;

Ejemplo de datos agrupado y ordenado:

A = LOAD 'data';
B = GROUP A BY $1;
C = FOREACH B {
      D = ORDER A BY ($3, $4);
      GENERATE D;
}
E = STREAM C THROUGH `stream.pl`;

Ejemplo de uso de esquema de formateo de datos:

X = STREAM A THROUGH `stream.pl` as (f1:int, f2:int, f3:int);
