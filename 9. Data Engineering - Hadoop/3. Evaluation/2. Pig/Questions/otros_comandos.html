Otros comandos

Grunt incluye también una serie de comandos para controlar Pig y Map Reduce: 

  • define: Permite definr funciones de usuario parametrizadas.
  • describe: Muestra el esquema definido para un determinado alias.
  • dump: Muestra el contenido de un alias de pig en la pantalla.
  • explain: Permite visualizar las etapas de ejecución de un determinado plan.
  • help: Lista los comandos disponibles.
  • illustrate: Muestra una ejecución de muestra de un script.
  • kill: Para un trabajo en curso en función de su ID.
  • quit: Sale de Grunt.
  • set:Permite parsear pares clave-valor a Pig de la forma set <key> '<value>'.
  • store: Permite almacenar el contenido de un alias de Pig a un fichero.

La descripción de cada uno de ellos se puede encontrar en el siguiente enlace: https://wiki.apache.org/pig/Grunt

A continuación se muestran algunos ejemplos de uso de los comandos anteriores describe e ilustrate sobre los discos de la sección anterior. En primer lugar carga el fichero discografía al alias discos mediante el comando:

discos = load 'ejemplos/discos' using PigStorage(',') as (anio, disco, eeuu, uk);
