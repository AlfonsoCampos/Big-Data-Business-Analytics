Grunt Shell

Grunt es el shell de comandos de Pig. El acceso al shell se realiza escribiendo pig -x local si lo se arranca en modo local o sólo pig si el caso de un cluster, como resultado muestra el prompt grunt>. Para salir hay que escribir quit o ejecutar Ctrl + D

 

Grunt incluye dos tipos de comandos: comandos de acceso al sistema de archivos DFS y comandos propios de Pig. A continuación se describen esos comandos, pero antes veremos un fichero de arranque que es muy útil para cargar algunas instrucciones por defecto.
