SAMPLE

Devuelve una muestra aleatoria basada en el tamaño indicado 

Devolver el 10% de un fichero 

A = LOAD 'data' AS (f1:int,f2:int,f3:int);

X = SAMPLE A 0.1;


En el siguiente ejemplo se emplea una función para devolver aproximadamente 1000 registros. 

a = load 'a.txt';
b = group a all;
c = foreach b generate COUNT(a) as num_rows;
d = sample a 1000/c.num_rows;
