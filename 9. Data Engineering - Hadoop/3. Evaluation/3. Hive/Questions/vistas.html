Vistas

Las vistas permiten guardar una consulta como si se tratara de una tabla. Es una construcción lógica que no almacena los datos como una tabla, es decir, no se materializa en una estructura en Hive. 

 

Cuando una query hace referencia a una vista, la información de su definición se combina con el resto de la query con el planificador de queries de Hive, es decir, se ejecuta por un lado la vista y después con el resultado se ejecuta el resto de la query. 

 

Las vistas se suelen emplear cuando las consultas son muy largas o complejas, de modo que se esconde la complejidad, dividiendo la query en una menor, más manejable. Encapsular la complejidad hace más fácil a los usuarios finales construir queries más manejables. 

Create View

Definición: permite crear una vista indicando un nombre. Si no se indican nombres de columna, entonces los nombres de las columnas de la vista se derivarán automáticamente de los definidos en la sentencia. La vista es un objeto puramente lógico sin almacenamiento asociado. 

Sintaxis: 

CREATE VIEW [IF NOT EXISTS] view_name [(column_name [COMMENT column_comment], ...) ]
[COMMENT view_comment]
[TBLPROPERTIES (property_name = property_value, ...)]
AS SELECT ...;

Ejemplo  

CREATE VIEW onion_referrers(url COMMENT 'URL of Referring page')
COMMENT 'Referrers to The Onion website'
AS
SELECT DISTINCT referrer_url
FROM page_view
WHERE page_url='http://www.theonion.com'; 

Drop View

Definición: permite eliminar los metadatos de una vista específica. Cuando se elimina una vista que hace referencia a otras vistas, no se muestra ningún aviso. 

Sintaxis:  DROP VIEW [IF EXISTS] view_name;

Ejemplo:  DROP VIEW paginas_vistas_jaen; 

Alter View Properties

Sintaxis:  permite añadir propiedades a una vista, del mismo modo que se pueden añadir a una tabla o a una partición. 

ALTER VIEW view_name SET TBLPROPERTIES table_properties;

table_properties:
: (property_name = property_value, property_name = property_value, ...)

Alter View As Select

Definición: modifica la definición de una vista si existía. La sintaxis es similar a la de Create View y sus efectos son similares.

Sintaxis: ALTER VIEW view_name AS select_statement;
