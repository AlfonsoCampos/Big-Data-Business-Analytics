Ejercicio 2: tipos de datos complejos

Partiendo de una base de datos de genios que contenga la siguiente información:

  • Nombre: Leonhard Paul
  • Apellido: Euler
  • Campos: ['física','matematicas']
  • Nacimiento:
    • Fecha: 1973-09-18
    • Ciudad: Basilea
    • País: Suiza

Realizar los siguientes ejercicios: 

1. Crear una base de datos que permita almacenar la estructura anterior

2. Insertar los siguientes elementos

{

      "nombre": "Leonhard Paul",

      "apellido": "Euler",              

      "campos": ["física", "matemáticas"],

      "nacimiento":{

          "fecha": new Date('1973-09-18'),

          "ciudad": "Basilea",

          "pais": "Suiza"

          }

     },

     {

      "nombre": "Johann Karl Friedrich",       

      "apellido": "Gauss",

      "campos": ["matemáticas", "astronomía", "geodesia", "física"],

      "nacimiento":{

          "fecha": new Date('1777-04-30'),

          "ciudad": "Brunswick",

          "pais": "Alemania"

          },

    },

    {

      "nombre": "Giuseppe Lodovico",

      "apellido": "Lagrange",

      "campos": ["física", "matemáticas", "astronomía"],

      "nacimiento":{

          "fecha": new Date('1736-01-25'),

          "ciudad": "Turín",

          "pais": "Italia"

      }

    },

    {

      "nombre": "Werner",

      "apellido": "Heisenberg",

      "campos": ["física"],

      "nacimiento":{

          "fecha": new Date('1901-12-05'),

          "ciudad": "Wurzburgo",

          "pais": "Alemania"

          }

    },

    {

      "nombre": "Roger",

      "apellido": "Penrose",

      "campos": ["física", "matemáticas"],

      "nacimiento":{

          "fecha": new Date('1931-08-08'),

          "ciudad": "Colchester",

          "pais": "Inglaterra"

      }

    }

3. Ejecutar una query que permita identificar cuantos elementos tienen como nombre "Euler"

4. Escribir una consulta para obtener los elementos de la tabla genios anterior que seleccione sólo aquellos personajes que hayan tratado los campos de las matemáticas y la geodesia (Pista: emplear una Collection Function de https://cwiki.apache.org/confluence/display/Hive/LanguageManual+UDF).

5. Escribir una consulta para obtener los elementos de la tabla genios anterior que seleccione sólo aquellos personajes que hayan tratado o bien el campo de la física o bien el campo de la geodesia, entre otros.

6. Escribir una consulta que devuelva el número de personajes nacidos en Alemania.

7. Escribir una consulta que devuelva los personajes nacidos en Alemania y que hayan trabajado en el campo de las matemáticas.

8. Escribir una consulta que devuelva el número de personajes que se han dedicado a la física como campo principal. Se entiende que los campos aparecen ordenados según dedicación.

9. Escribir una consulta que devuelva únicamente el nombre de los personajes que cumplen el criterio del ejercicio anterior.
