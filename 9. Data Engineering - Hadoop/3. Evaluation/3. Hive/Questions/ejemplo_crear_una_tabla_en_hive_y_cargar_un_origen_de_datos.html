EJEMPLO: Crear una tabla en Hive y cargar un origen de datos

Partiendo de los datos estadísticos del gobierno, realizaremos una carga de datos de un fichero en Hive y después efectuaremos una serie de consultas sobre él.

 

1. Obtener el fichero con los datos a analizar.

Para ello accederemos a la siguiente página

http://www.minhap.gob.es/es-ES/Areas%20Tematicas/Administracion%20Electronica/OVEELL/Paginas/DeudaViva.aspx

 

y descargaremos el fichero del año 2012.

 

 

 

Se nos abrirá una página de descarga y seleccionaremos la opción guardar. 

 

 

 

 

A continuación abriremos el fichero 

 

 

Se nos abrirá un fichero como el siguiente con una información por municipios y deudas asociadas. Nos centraremos en la solapa Datos_Format. 

 

 

 

2. Preparar el fichero para su procesamiento

 

Lo siguiente que haremos es preparar el fichero para poder procesarlo. En la mayoría de los ejercicios de procesamiento de datos es necesario realizar primero una fase de limpieza y preparación de los datos.

 

En este caso sólo tendremos que eliminar las 5 primeras líneas para quedarnos sólo con las filas que poseen datos de análisis 

 

así como las últimas filas. 

 

y formatearemos los importes de deudas eliminando los separadores de miles. 

y lo guardaremos con la opción GUARDAR COMO

 

 

como fichero csv en la ruta /tmp con el nombre Deuda2012.csv

 

 

3. Arrancamos Hive

 

Para ello arancaremos HDFS, MapReduce y después escribiremos el comando

 

$./bin/hive o $hive

 

 

 

4. Crear la tabla deuda_2012 en la que almacenaremos la información del fichero: 

 

cod_comunidad: String

cod_prov: String

cod_municipio: String

municipio: String

deuda: int

 

 

CREATE TABLE deuda_2012(cod_comunidad STRING, cod_prov STRING, cod_municipio STRING, municipio STRING, deuda INT)

ROW FORMAT DELIMITED FIELDS TERMINATED BY '\;';

 

 

 

5. Cargar el fichero Deuda2012.cvs en la tabla anterior

 

LOAD DATA LOCAL INPATH '/tmp/Deuda2012.csv' INTO TABLE deuda_2012;

 

 

6. Ejecuta una select sin criterios de filtrado para consultar información de la tabla: 

 

SELECT deuda_2012.* FROM deuda_2012;

 

 

7. Ejecuta una sentencia que permita filtrar algunos datos. 

 

SELECT * FROM deuda_2012 WHERE municipio LIKE 'Madri%';
