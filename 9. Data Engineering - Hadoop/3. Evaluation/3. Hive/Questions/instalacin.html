Instalación

Prerrequisitos 

 

  • Java 1.6 o Java 1.7 (preferentemente)

  • Hadoop  2.x (preferentemente), 1.x.  

  • Hive se suele emplear en entornos de producción con Linux o Windows. Mac se suele emplear como entorno de desarrollo. 

 

Descarga

 

Para poder instalar Hive, procederemos a seleccionar la última versión accesible desde sus ventanas de descarga: 

 

https://hive.apache.org/downloads.html 

http://www.apache.org/dyn/closer.cgi/hive/

 

Seleccionaremos la última versión estable

 y la descargaremos con la siguiente instrucción

wget http://apache.rediris.es/hive/stable/apache-hive-0.14.0-bin.tar.gz

 

 

 

Después procederemos a descomprimir el paquete

 

$ tar -xvf apache-hive-0.14.0-bin.tar.gz

 

Moveremos el fichero a la ruta correspondiente

 

$ mv apache-hive-0.14.0-bin /home/bigdata/hive 

 

 

Establecemos la variable de entorno HIVE_HOME que apunte al directorio de la instalación y la ruta de los binarios al path

sudo nano ~/.bashrc

# Configuración de HIVE

export HIVE_HOME=/home/bigdata/hive

export PATH=$HIVE_HOME/bin:$PATH
# Fin configuración de HIVE

source ~/.bashrc

Arrancamos Hive

 

Antes de arrancar Hive, debemos arrancar Hadoop y además es necesario crear las siguientes estructuras de directorios con permisos g+w antes de crear una tabla en Hive: 

 

$ start-dfs.sh

$ jps

$ $HADOOP_HOME/bin/hdfs dfs -mkdir       /tmp

$ $HADOOP_HOME/bin/hdfs dfs -mkdir       /bigdata

$ $HADOOP_HOME/bin/hdfs dfs -mkdir       /bigdata/hive
$ $HADOOP_HOME/bin/hdfs dfs -mkdir       /bigdata/hive/warehouse
$ $HADOOP_HOME/bin/hdfs dfs -chmod g+w   /tmp
$ $HADOOP_HOME/bin/hdfs dfs -chmod g+w   /bigdata

$ $HADOOP_HOME/bin/hdfs dfs -chmod g+w   /bigdata/hive

$ $HADOOP_HOME/bin/hdfs dfs -chmod g+w   /bigdata/hive/warehouse

 

Por último arrancamos Hive escribiendo $HIVE_HOME/bin/hive o sólo hive al haberlo incluido en el path. 

 

Para salir de Hive emplearemos el comando quit;
