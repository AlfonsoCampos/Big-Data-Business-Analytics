Import-all-tables

Con los comandos anteriores, hemos visto cómo se podía migrar una tabla individual. Mediante el comando import-all-tables podemos migrar una base de datos completa. 

En este caso la sintaxis es muy parecida pero algunos comandos dejan de tener sentido, como por ejemplo los comandos --table, --where, --columns, y aparecen nuevos comandos específicos de las migraciones masivas como es el comando --exclude-tables <table_names>.

Ejemplo: 

$ sqoop import-all-tables --connect jdbc:mysql//localhost/mydb  --username usuario
