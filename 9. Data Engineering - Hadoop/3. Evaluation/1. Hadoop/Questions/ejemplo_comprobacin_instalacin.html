Ejemplo comprobación instalación

En esta sección se presenta un ejercicio que comprobará la correcta instalación del sistema. El ejercicio consistirá en:

  1. Creación de un directorio.
  2. Creación de un archivo de ejemplo.
  3. Inclusión del directorio creado en el sistema de archivos HDFS.
  4. Ejecución de un trabajo que cuente el numero de repeticiones de cada palabra existente en el fichero.

 

A  continuación se detalla el ejemplo de wordcount para verificar la instalación.

Primero se creará el árbol de directorios donde se ubicará el archivo a tratar:

mkdir /home/bigdata/hadoop/ejercicios

mkdir /home/bigdata/hadoop/ejercicios/ejercicio1

mkdir /home/bigdata/hadoop/ejercicios/ejercicio1/in


Se crea un archivo llamado "ejemplo.txt" y se escriben dos líneas de ejemplo.

cd /home/bigdata/hadoop/ejercicios/ejercicio1/in

cat > ejemplo.txt
Esta es una línea
Esta es otra línea de ejemplo

CTRL+C finalizará la escritura.

Como resultado se habrá creado un fichero con esa información.

ls

Una vez creado el fichero, se añade a HDFS:

cd ..

hdfs dfs -copyFromLocal in /in

Con el siguiente comando se comprobará si se ha añadido el directorio a HDFS de manera correcta.

hadoop fs -ls /

Para eliminar un directorio se emplea la instrucción

hdfs dfs -rm -r hdfs:/in

Se vuelven a copiar los ficheros, pero esta vez en una ruta del directorio ejercicios1

hdfs dfs -copyFromLocal in hdfs:/ejercicio1

hadoop fs -ls /

Ejecución del comando word count:

hadoop jar /home/bigdata/hadoop/share/hadoop/mapreduce/hadoop-mapreduce-examples-2.6.0.jar wordcount /ejercicio1 /out
