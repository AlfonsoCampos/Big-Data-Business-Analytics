Confirmar versión de Java

Hadoop requiere que la versión de Java 1.6 (o superior) esté instalada, no obstante algunas herramientas que serán utilizadas requieren la verisón 1.7, por lo que se comprobará que esté instalada ésta última.

java -version

Una vez se confirme que la versión Java 1.7 se encuentre correctamente instalada, se busca la ruta en la que se encuentra instalada Java y se comprueba la variable de entorno JAVA_HOME que será requerida en pasos posteriores.

sudo update-alternatives --config java

Por lo tanto, el path completo es /usr/lib/jvm/java-7-openjdk/jre/bin/java.

El valor de la variable JAVA_HOME es todo lo que se encuentra antes de /jre/bin/java, que es por lo tanto /usr/lib/jvm/java-7-openjdk-amd64.

OPCIONAL: En el caso de que no esté instalado, se ejecutan los siguientes comandos.

sudo apt-get install openjdk-7-jre 

Si se ha tenido que instalar java, se debe ejecutar nuevamente el siguiente comando para verificar que está instalado correctamente y que la versión es, al menos, la 1.7.

java -version

Obteniendo una información similar a la siguiente:

Como no está correctamente instalada la versión requerida, se debe ejecutar el siguiente comando para seleccionar la versión correcta.

sudo update-alternatives --config java
