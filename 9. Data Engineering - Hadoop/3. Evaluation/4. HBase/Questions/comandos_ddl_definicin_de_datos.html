Comandos DDL (definición de datos)

Operaciones sobre NameSpaces: 

  • create: permite crear namespaces 
    • create_namespace 'master2015'

  • alter_namespace: permite añadir o eliminar una propiedad de un namespace
    • alter_namespace 'master2015', {METHOD=>'set','Propiedad1'=>'Valor1'}
    • alter_namespace 'master2015', {METHOD=>'set','Propiedad2'=>'Valor2'}

    • alter_namespace 'master2015', {METHOD=>'unset',NAME=>'Propiedad2'}

  • describe_namespace: muestra la información del namespace
    • describe_namespace 'master2015'

  • list_namespace: permite mostrar la información de los tablespaces existentes
    • list_namespace 

Operaciones sobre Tablas:

  • create: crea una nueva tabla en Hbase
    • Ejemplo 1: namespace = master2015
      • create 'master2015:tabla_ejemplo',  {NAME => 'column_family_1', VERSIONS => 1, TTL => 2592000, BLOCKCACHE => true}

    • Ejemplo 2: namespace = default
      • create 'tabla_ejemplo_en_default_namespace',  {NAME => 'column_family_1'}, {NAME => 'column_family_2'}, {NAME => 'column_family_3'}

      • create 'tabla1', 'column_family_1', 'column_family_2', 'column_family_3'

  • alter: permite modificar la información de una tabla así como atributos como MAX_FILESIZE, MEMSTORE_FLUSHSIZE, READONLY, y DEFERRED_LOG_FLUSH.
    • Ejemplo 1: Cambiar o añadir la columna Family column_family_1 en la tabla tabla1 manteniendo un máximo de 5 versiones de Celdas.
      • hbase> alter 'tabla1', NAME => 'column_family_1', VERSIONS => 5

    • Ejemplo 2: Cambiar el tamaño máximo de la column family a 128MB
      • hbase> alter 'tabla1', METHOD => 'table_att', MAX_FILESIZE => '134217728'

    • Ejemplo 3: Cambiar varios parámetros de una vez
      • hbase> alter  'tabla1', {NAME => 'column_family_1'}, {NAME => 'column_family_2', METHOD => 'delete'} 

  • delete: permite borrar elementos de una tabla
    • Ejemplo 1: borrar la column family 1 de la tabla 1
      • hbase> alter 'tabla1', NAME => 'column_family_1', METHOD => 'delete'

    • Ejemplo 2: de un modo más simplificado
      • hbase> alter 'tabla1', 'delete' => 'column_family_3' 

  • describe: muestra la descripción de la tabla
    • Ejemplo: describe 'tabla1'

  • disable : deshabilita una tabla. Por ejemplo es necesario deshabilitarla para poder borrarla. 
    • Ejemplo: 
      • hbase> disable 'tabla1'

  • drop : Elimina una tabla. Antes de eliminarla hay que deshabilitarla. Si la tabla tiene más de una región hay que ejecutar antes el comando de compactación en .META.
    • Ejemplo:
      • hbase> drop 'tabla1'

  • enable : Habilita una tabla
    • Ejemplo
      • hbase> create 't1', 'col1', 'col2', 'col3'
      • hbase> disable 't1'
      • hbase> enable 't1'

  • exists : Comprueba si una tabla existe
    • Ejemplo:
      • hbase> exists 't1'

  • is_disabled : Comprueba si una tabla está deshabilitada
    • Ejemplo:
      • hbase> is_disabled 't1'

  • is_enabled : Comprueba si una tabla está habilitada
    • Ejemplo:
      • hbase> is_enabled 't1'

  • list : Lista todas las tablas de una base de datos.
    • Ejemplo:
      • hbase> list

  • truncate : deshbilita, elimina o recrea una tabla.
    • Ejemplo:
      • hbase> truncate 't1'
