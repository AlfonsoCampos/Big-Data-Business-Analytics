Comandos de manipulación de datos CRUD.

Comandos de manipulación de datos sobre filas y columnas

  • put: inserta el valor de una celda en la tabla/fila/columna especificada y opcionalmente con las coordenadas del timestamp.
    • Ejemplo: poner una celda con valor 1 en la tabla 1  en la fila 1 en la columna 1 
      • put 't1', 'r1', 'col1', '1' 

      • put 't1', 'r2', 'col1', 1
  • get: recupera los contenidos de una celda o fila.
    • Ejemplos:
      • hbase> get 't1', 'r1'

    • Si se dispone de un conjunto de datos más extenso, se pueden emplear las siguientes consultas:
      • hbase> get 't1', 'r1', {TIMERANGE => [ts1, ts2]}
      • hbase> get 't1', 'r1', {COLUMN => 'col1'}
      • hbase> get 't1', 'r1', {COLUMN => ['col1', 'col2', 'col3']}
      • hbase> get 't1', 'r1', {COLUMN => 'col1', TIMESTAMP => ts1}
      • hbase> get 't1', 'r1', {COLUMN => 'col1', TIMERANGE => [ts1, ts2], VERSIONS => 4}
      • hbase> get 't1', 'r1', {COLUMN => 'col1', TIMESTAMP => ts1, VERSIONS => 4}
      • hbase> get 't1', 'r1', 'col1'
      • hbase> get 't1', 'r1', 'col1', 'col2'
      • hbase> get 't1', 'r1', ['col1', 'col2'] 
  • incr: aumenta el valor de una celda, indicando sus coordenadas (tabla/fila/columna)
    • Ejemplos:
      • hbase> incr 't1', 'r1', 'col1' → incrementar en 1
      • hbase> incr 't1', 'r1', 'col1', 1→ incrementar en 1
      • hbase> incr 't1', 'r1', 'col1', 10→ incrementar en 10

  • count: devuelve el nº de filas de una tabla.Esta operación tardará mucho. Por defecto, count se muestra cada 1000 filas por defecto, pero el intervalo puede ser indicado excepcionalmente, del mismo modo que los scaneos de caché se pueden indicar.
    • Ejemplos:
      • hbase> count 't1'

      • hbase> count 't1', INTERVAL => 100000
      • hbase> count 't1', CACHE => 1000
      • hbase> count 't1', INTERVAL => 10, CACHE => 1000 
  • getcounter: devuelve el contador de una celda especificando las coordenadas de la tabla/fila/columna.
    • Ejemplos:
      • hbase> get_counter 't1', 'r1', 'col1'
  • delete: elimina una celda de una tabla/fila/columna
    • Ejemplo:
      • hbase> delete 't1', 'r1', 'col1'

  • delete_all: elimina toda una fila. Opcionalmente se puede indicar la columna y el timestamp
    • Ejemplos:
      • hbase> deleteall 't1', 'r1'

      • hbase> deleteall 't1', 'r1', 'c1'
      • hbase> deleteall 't1', 'r1', 'c1', ts1
